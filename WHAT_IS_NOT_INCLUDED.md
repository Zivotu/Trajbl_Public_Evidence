# What Is Not Included

To protect our intellectual property, this public repository contains only aggregate metrics, conceptual methodologies, and public demo instructions.

---

## Excluded Proprietary Assets

The following files and components are strictly excluded from this repository:

1. **Source Code:**
   * No selector scripts or routing implementation files.
   * No tagger implementation scripts or matching heuristics files.
   * No adapter, runtime, or gateway configs.
   * No binary, library, or compiled core modules.

2. **Scoring Logic & Parameters:**
   * No specific cue listings or dictionary structures.
   * No scoring boundaries, tuning coefficients, or algebraic variables used in sentence ranking.

3. **Data Assets:**
   * No private benchmark datasets or raw evaluation transcripts.
   * No reference answer sheets or reference texts.
   * No structured index databases.

4. **Prompt Assets:**
   * No prompt logs or templates showing internal LLM system instructions.

---

## NDA and Partnership Access

Partners interested in verifying the underlying codebase, running local reproductions, or evaluating the full scoring implementation can request review access. Source code review and detailed parameter lists are shared only under a Non-Disclosure Agreement (NDA) as part of our formal partnership process.
